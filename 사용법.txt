  인증서 이사 도우미 - 사용법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 실행
  「인증서 이사 도우미.exe」(파란 체크 아이콘) 더블클릭.
  설치할 것 없고, 실행하면 창이 하나 열리며 인증서를 자동으로 찾아 줍니다.
  ※ 파란 "Windows의 PC 보호" 창이 뜨면 [추가 정보] → [실행]

■ 인증서를 다른 컴퓨터로 옮기기 (이사)
  【보내는 컴퓨터】
   1. USB에 이 프로그램 폴더를 통째로 복사 → USB에서 exe 실행
   2. [인증서 가져가기] → 옮길 인증서를 눌러 선택 → [백업 만들기]
   3. 끝. 백업은 프로그램 옆 [백업] 폴더에 저장 = 이미 USB 안
  【받는 컴퓨터】
   4. USB 꽂고 exe 실행 → [인증서 설치하기]에 백업이 자동으로 떠 있음
   5. 내용 확인 → [이 컴퓨터에 설치] → 끝
  비밀번호는 인증서 안에 있으므로 쓰던 그대로 입력하면 됩니다.

■ 필요없는 파일 정리
  [정리하기] → 인증서 폴더 속 잡동사니(압축파일·문서 등)와
  만료·중복 인증서, 빈 폴더를 자동으로 골라 줍니다.
  [휴지통으로 보내기]를 누르면 휴지통으로 가므로,
  실수했으면 휴지통에서 복원하면 됩니다.

■ 인증서 메모
  각 인증서의 [✏ 메모]로 메모를 남길 수 있습니다.
  (예: "학교회계용", "OO은행", "2027년 갱신")
  메모는 백업에 같이 담겨 옮긴 컴퓨터에서도 보입니다.

■ 관리자용 (화면 맨 아래)
  · 규칙 설정: 인증서를 찾는 위치·파일 구조·잡파일 판정·설치 위치를
    JSON으로 직접 수정. 새 사례를 만나면 프로그램 수정 없이 여기만 고침.
  · 로그: 화면에서 바로 보기. 파일은 logs\ 폴더에 실행마다 저장.
    문제가 생기면 logs 폴더째 관리자에게 전달.

■ 주의
  인증서는 도장과 같습니다. 백업은 본인 컴퓨터 이사 용도로만 쓰고,
  전달이 끝나면 USB·메일에서 백업파일을 지우세요.
