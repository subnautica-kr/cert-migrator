# 🔐 인증서 이사 도우미

공동인증서(NPKI)·행정전자서명(GPKI)·교육기관 인증서(EPKI)를 **컴맹도 클릭 몇 번으로**
다른 컴퓨터로 옮기고, 지저분한 인증서 폴더를 정리하는 무설치 도구입니다.

경로가 PC마다 다르고(`AppData\LocalLow`, `Program Files`, `C:\GPKI`, USB 등),
인증서 프로그램마다 저장 구조가 달라서 공유가 번거로운 문제를 해결합니다.

made by **subnautica-kr** · 경상남도교육청 소속 전산주무관 개인 제작
(경상남도교육청의 공식 프로그램이 아니며, 기관의 지원·보증과 무관합니다.)

---

## 특징
- **무설치**: Windows 기본 PowerShell + Edge만 있으면 실행. 관리자 권한 불필요
- **자동 검색**: NPKI/GPKI/EPKI를 알려진 위치 전부에서 자동으로 찾음
- **정체 식별**: 발급기관·용도(은행/증권/공무원/교육) 표시
- **안전한 정리**: 잡파일·만료·중복을 **휴지통**으로 (삭제 아님, 복구 가능)
- **인증서 메모**: 어디에 쓰는 인증서인지 메모, 백업에 함께 이동
- **규칙 기반**: 탐색·설치·정리 기준이 `app/rules.json` — 새 사례는 여기만 수정
- **로그 수집**: 실행 기록이 `logs/`에 남아 원격 유지보수 가능

## 사용법
1. 이 폴더를 통째로 USB에 복사
2. `인증서 이사 도우미.exe` 더블클릭
3. 보내기: **인증서 가져가기** → 선택 → 백업 만들기 (프로그램 옆 `백업/`에 저장)
4. 받기: 다른 PC에서 실행 → **인증서 설치하기** (백업 자동 인식) → 설치

자세한 내용은 [사용법.txt](사용법.txt), 변경 내역은 [CHANGELOG.md](CHANGELOG.md) 참고.

## 폴더 구조
```
인증서 이사 도우미.exe   # 실행 파일(런처)
app/
  main.ps1              # 엔진 + 로컬 UI 서버
  rules.json            # 탐색·정리·설치 규칙 (화면에서도 수정 가능)
  ui/                   # 화면 (index.html / style.css / app.js / assets)
사용법.txt
CHANGELOG.md · LICENSE · version.json
```
`logs/`, `백업/`, `app/memos.json` 은 실행 중 생성되며 저장소에 포함하지 않습니다.

## 업데이트
프로그램 [정보] 화면의 **업데이트 확인**이 GitHub의 `version.json`을 확인해
새 버전이 있으면 알려줍니다. (확인 주소는 `app/rules.json`의 `업데이트.확인URL`)

## 개발 / 배포 메모
- 화면(HTML/CSS/JS)만 고칠 땐 `app/ui/*` 편집 후 창 새로고침이면 끝 (exe 재빌드 불필요)
- 엔진 로직은 `app/main.ps1`
- PowerShell 5.1 기준. UI는 Edge `--app` 모드로 렌더링
