# 변경 이력 (Changelog)

이 프로그램의 버전별 변경 내용입니다. 위쪽이 최신입니다.

## v2.6 (2026-07-22)
- 정보 화면 정비: 제작자 표기(made by subnautica-kr · 경상남도교육청, 개인 제작 명시)
- 라이선스 CC BY-NC 4.0 (출처 유지 + 비상업)
- 자동 업데이트: 새 버전이 확인되면 GitHub에서 받아 자동 설치·재시작,
  확인 불가 시 구석에 "최신버전 확인할 수 없음"만 표시(사용은 정상)

## v2.5 (2026-07-22)
- 뒤로가기 지원: 마우스 뒤로/앞으로 버튼, `Alt+←/→`, `백스페이스`, `Esc`
- 모달창에서 `Enter`=확인, `Esc`=취소
- 작업표시줄·창 아이콘을 전용 아이콘(파란 체크)으로 표시 (favicon 적용)
- 정보(About) 화면 추가: 버전·라이선스·변경이력·제작자·업데이트 확인
- 진단용 임시 파일 정리

## v2.4 (2026-07-22)
- **무한로딩 버그 수정**: 숨김 처리한 로딩 오버레이가 CSS 충돌로 계속 화면에 남아
  클릭을 막던 문제 해결 (`[hidden]` 강제 적용). 서버·소켓은 정상이었음

## v2.3 (2026-07-22)
- "인증서 도우미" → **"인증서 이사 도우미"** 로 이름 변경
- 로컬 서버가 브라우저의 빈 연결(preconnect)에 막혀 멈추던 문제 방지
- CSS/JS를 한 번의 요청으로 합쳐 전송(정체 방지), 요청 로그 추가

## v2.2 (2026-07-22)
- 화면을 웹(HTML/CSS/JS) 기반 카드 UI로 전면 개편 (Edge 앱모드)
- 원페이지 → 홈에서 기능별 화면으로 분리

## v2.1 (2026-07-22)
- **안전장치**: 정리 시 인증서 폴더를 통째로 지우지 않고 해당 인증서 파일만 처리
  (한 폴더에 여러 사람 인증서가 섞인 GPKI에서 남의 인증서가 딸려가던 사고 방지)
- 비어 있어도 지우면 안 되는 표준 폴더(USER/class1 등) 보호

## v2.0 (2026-07-22)
- 규칙(rules.json) 기반으로 탐색·정리·설치·백업을 통제하도록 재설계
- 정리 = 삭제 대신 **휴지통 이동**
- 인증서 메모 기능
- 폴더형(NPKI) + 파일묶음형(GPKI/EPKI) 두 저장방식 모두 지원

## v1.x (2026-07-22)
- 최초 버전: NPKI/GPKI/EPKI 자동 검색, 백업(zip) 만들기, 설치, 로그 수집
- 인증서 종류·발급기관·용도 식별, 기관CA 캐시 자동 제외
