# 변경 이력 (Changelog)

이 프로그램의 버전별 변경 내용입니다. 위쪽이 최신입니다.

## v2.9 (2026-07-23)
- 인증서 목록을 조밀하게(항목 높이 대폭 축소) — 한 화면에 더 많이 보임
- 변경이력을 버전 카드(배지·날짜·최신 표시)로 보기 좋게
- 규칙 편집기 글씨 축소, 로그를 터미널(cmd)풍으로
- 창 폭 1080으로 넓힘. 단 로그·변경이력·규칙 화면만 넓게 쓰고 나머지는 읽기 좋은 폭 유지

## v2.8 (2026-07-23)
- **WebView2 네이티브 창**: 작업표시줄·창 아이콘이 전용 아이콘으로, Edge 흔적 제거
  (실행파일이 창의 주인. WebView2 런타임은 Windows 11 기본 탑재)
- 창 크기 380×600(세로형)으로 축소, 여백·불필요한 스크롤 정리
- "최신버전 확인할 수 없음"은 오른쪽 위에 작게 떴다가 4초 뒤 사라짐
- **자동 업데이트를 교육청 방화벽 대응 방식으로 변경**: CDN(raw)이 막혀도
  github.com/codeload는 열려 있어, tags.atom으로 최신 태그 확인 + codeload로
  태그 zip 다운로드(curl/schannel). 새 태그를 올리면 자동 배포됨

## v2.6 (2026-07-22)
- 정보 화면 정비: 제작자 표기(made by subnautica-kr · 경상남도교육청, 개인 제작 명시)
- 라이선스 CC BY-NC 4.0 (출처 유지 + 비상업)
- 자동 업데이트: 새 버전이 확인되면 GitHub에서 받아 자동 설치·재시작,
  확인 불가 시 구석에 "최신버전 확인할 수 없음"만 표시(사용은 정상)

## v2.5 (2026-07-22)
- 뒤로가기 지원: 마우스 뒤로/앞으로 버튼, `Alt+←/→`, `백스페이스`, `Esc`
- 모달창에서 `Enter`=확인, `Esc`=취소
- 작업표시줄·창 아이콘을 전용 아이콘(파란 체크)으로 표시 (favicon 적용)
- 정보(About) 화면 추가: 버전·라이선스·변경이력·제작자·업데이트 확인
- 진단용 임시 파일 정리

## v2.4 (2026-07-22)
- **무한로딩 버그 수정**: 숨김 처리한 로딩 오버레이가 CSS 충돌로 계속 화면에 남아
  클릭을 막던 문제 해결 (`[hidden]` 강제 적용). 서버·소켓은 정상이었음

## v2.3 (2026-07-22)
- "인증서 도우미" → **"인증서 이사 도우미"** 로 이름 변경

- CSS/JS를 한 번의 요청으로 합쳐 전송(정체 방지), 요청 로그 추가

## v2.2 (2026-07-22)

## v2.1 (2026-07-22)
